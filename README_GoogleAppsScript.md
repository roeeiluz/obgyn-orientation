# אירוח וסנכרון דרך Google Apps Script (מתאים לרשת רמב"ם)

רשת רמב"ם חוסמת את GitHub (כולל `github.io`), אבל מאפשרת את `google.com`. לכן
האפליקציה מתארחת דרך **Google Apps Script** שמשמש כ"מתווך":

1. **מגיש את האפליקציה** — מושך את `index.html` העדכני מ-GitHub בכל טעינה.
   המשיכה קורית בשרת של גוגל (שכן מגיע ל-GitHub), כך שהדפדפן שלך מדבר רק עם
   google.com. כל `push` של קלוד ל-GitHub מופיע אוטומטית — **בלי העתק-הדבק**.
2. **שומר את הנתונים** המשותפים, לסנכרון חי בין מכשירים (עדכון כל ~5 שניות).

## תרשים הזרימה
```
קלוד עורך  →  push ל-GitHub (main)
                     │  (משיכה בשרת גוגל — לא חסום)
                     ▼
         Apps Script /exec  ──►  הנייד שלך ברמב"ב (דרך google.com)
                     ▲
                     └── שמירת/טעינת נתונים (?api=state)
```

---

## הקמה (פעם אחת, ~5 דקות)

### שלב 1 — הקוד
1. [script.google.com](https://script.google.com) → **New project**.
2. מחק/י את התוכן ב-`Code.gs`, הדבק/י את כל תוכן `apps_script/Code.gs` מכאן, ושמור/י.

### שלב 2 — פריסה כ-Web App
1. **Deploy → New deployment** → גלגל שיניים → **Web app**.
2. **Execute as:** `Me` · **Who has access:** `Anyone`  ← חובה (אחרת 403).
3. **Deploy** → אשר/י הרשאות (חשבון Google שלך → Advanced → Go to project → Allow).
4. העתק/י את ה-**Web app URL** שמסתיים ב-`/exec`.

### שלב 3 — חיבור (פעם אחת)
הכתובת כבר מוגדרת ב-`index.html` תחת `CONFIG.GAS_URL` (עם `?api=state`). אם
פרסת/ה כתובת חדשה משלך — עדכן/י אותה שם, וגם את `APP_URL` בראש `Code.gs` אם
שינית repo/ענף.

**זהו.** מרגע זה אתה פותח בנייד את כתובת ה-`/exec`, והאפליקציה תמיד עדכנית.

## שגרת העבודה מכאן
- **לשנות את האפליקציה:** מבקש מקלוד → קלוד דוחף ל-GitHub (`main`) → מרענן בנייד.
  **לא נוגעים ב-Apps Script.** (עדכון מ-GitHub raw עשוי לקחת עד ~5 דק' בגלל
  קאשינג; רענון נוסף פותר.)
- **לשנות את ה-backend עצמו** (`Code.gs`, נדיר): Deploy → Manage deployments →
  Edit → Version: New version → Deploy. כך הכתובת נשארת זהה.

## אבחון תקלות
- **403 / דף התחברות:** ה-Web app לא "Anyone". חזור/חזרי לשלב 2.2.
- **דף ריק / "שגיאה בטעינת האפליקציה":** בעיית משיכה מ-GitHub — ודא/י ש-`APP_URL`
  נכון ושה-repo ציבורי.
- **נתונים לא מסונכרנים:** F12 → קונסול → חפש/י `gas push` / `gas pull`.

## הערות
- **מצב נתונים בלבד (בלי הגשת אפליקציה):** אם תארח/י את ה-HTML במקום אחר, אפשר
  להשתמש באותו סקריפט רק ל-`?api=state` ולהתעלם מ-`serveApp`.
- **קישורי PDF (פרוטוקולים/ספרים):** קישורים יחסיים לא ייפתחו דרך ההגשה הזו —
  שדרוג נפרד (העלאה ל-Google Drive וקישור ישיר) בהמשך.
- **אבטחה:** אין מידע רפואי של מטופלים. הכתובת ציבורית — מי שיש לו אותה יכול
  לקרוא ולכתוב. תואם למודל מבוסס-האמון של התוכנית.
